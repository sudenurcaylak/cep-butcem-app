# 📱 Cep Bütçem

A full-featured mobile budget tracking application built with Flutter, designed to help users manage their personal finances efficiently.

## 🚀 Features

- Track income and expenses
- Analyze spending habits with visual insights
- Manage multiple accounts (bank, wallet, etc.)
- Add and organize custom categories
- Subscription management
- Reminder system
- Transaction history
- Offline-first architecture
- Modern and responsive mobile UI

## 🛠 Tech Stack

- Flutter
- Dart
- SQLite (sqflite)
- go_router
- Repository Pattern

## 📊 Project Highlights

- Local database management with SQLite
- Scalable and maintainable project architecture
- Custom UI/UX design
- Offline data persistence
- Financial analytics and reporting

## 🎯 Purpose of the Project

Cep Bütçem was developed to provide users with a simple, fast, and effective way to track their personal finances directly from their mobile devices.

The project focuses on mobile application architecture, local database management, and user-friendly financial tracking experiences.
